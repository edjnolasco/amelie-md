# Introducción

Este proyecto demuestra el modo de publicación por proyecto introducido en Amelie MD v1.6.0.

:::definition def-project Proyecto editorial
Un proyecto editorial organiza capítulos, referencias, configuración y salidas dentro de una misma unidad de publicación.
:::

Como se explica en {{ref:def-project}}, el documento deja de depender de un único archivo Markdown y pasa a operar como un workspace estructurado.
