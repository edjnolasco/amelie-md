# Amelie MD v1.3.0 — Semantic Academic Publishing

Release Date: 2026-05-13

## Overview

Amelie MD v1.3 represents the transition from a Markdown-oriented export utility into a semantic academic publishing engine.

This release introduces structured semantic authoring, academic semantic rendering, semantic references, chapter-aware numbering, semantic indexes, and cross-render semantic publishing support for HTML and DOCX outputs.

The v1.3 architecture formalizes semantic publishing as a first-class pipeline inside Amelie MD.

# Major Features

## Semantic Academic Blocks
