# Changelog

## v1.3.0 — Semantic Academic Publishing

### Added

- Semantic academic blocks:
  - Definitions
  - Figures
  - Quotes
  - Admonitions
- Semantic authoring syntax using `:::block`
- Semantic normalization pipeline
- Chapter-aware semantic numbering
- Semantic cross references
- Internal HTML hyperlinks
- Semantic indexes:
  - List of Figures
  - List of Definitions
- Semantic contracts and validation
- Central semantic taxonomy
- Central semantic pipeline orchestration
- Canonical semantic showcase document
- Cross-render semantic parity for HTML and DOCX

### Improved

- Renderer registry architecture
- Component rendering organization
- Semantic block ingestion
- Academic navigation support
- Internal semantic identity handling

### Architecture

The v1.3 release transforms Amelie MD from a Markdown-oriented exporter into a semantic academic publishing engine with structured semantic rendering and academic navigation support.
